AutoHotkey.net Tool

The AutoHotkey.net Tool provides you an easy way to upload your scripts to AutoHotkey.net from the editor. It consists of a set of special comments that will tell the tool which files to upload, etc.

First you have to write a series of lines in your AHK file. This is the syntax for the lines (semicolon included):

;#AHKNet_CommandName parameter
;#AHKNet_CommandName "param1" "param2" ...

The commands are:

UploadScript

Uploads the script to the specified location.

;#AHKNet_UploadScript Location
LocationThe location on AutoHotkey.net. The directory is optional (if omitted it defaults to the current remote directory), but you must specify the filename.

If any of the folders do not exist, they are created.

Example:

;#AHKNet_UploadScript Functions/Add.ahk

UploadFile

Uploads a file to the specified location.

;#AHKNet_UploadFile "LocalFile" "RemoteLocation"
LocalFileThe file to upload.
RemoteLocationThe location on AutoHotkey.net. Refer to UploadScript's parameter for more information.

Both parameters must be enclosed in quotes.

If any of the remote folders do not exist, they are created.

Example:

;#AHKNet_UploadFile "library.dll" "dependencies/library.dll"

ChangeDirectory

Changes the current remote directory (similar to MS-DOS cd).

;#AHKNet_ChangeDirectory Location
LocationThe directory to change.

Example:

;#AHKNet_ChangeDirectory MyProjectFolder

Here is a more detailed example that shows how to upload a script and its dependencies:

; Upload this script to myScripts/someScript
;#AHKNet_UploadScript myScripts/someScript/main.ahk

; Go to myScripts/someScript. Directory checking takes a lot of time
; so it's better to check only once (UploadScript), go there and upload
; the files without specifying the path.
;#AHKNet_ChangeDirectory myScripts/ahknetutil

; Upload the dependencies
;#AHKNet_UploadFile "dependency1.ahk" "dependency1.ahk"
;#AHKNet_UploadFile "dependency2.ahk" "dependency2.ahk"
;#AHKNet_UploadFile "dependency3.ahk" "dependency3.ahk"

In order to execute the commands, open your script then click on the AutoHotkey.net Tool icon in the toolbar:

Fill in the fields (or leave them if you previously told the program to remember your data) then click on Execute.
