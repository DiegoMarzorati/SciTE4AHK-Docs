Changing settings

If you want to quickly change some specific settings, go to Tools > "SciTE4AutoHotkey settings...".

Here you can change the display language, the syntax highlighting style, the default script codepage, the default zoom and turn on/off the auto-backup feature and window position remembering.

However, there are many more settings to customize. They must be configured using SciTE property files. The recommended file to edit is the User properties file (SciTEUser.properties). You can open it by going to Options > "Open User properties". SciTE property lines take this format:

# This is a comment
variable.name=value

There MUST NOT be spaces between the equals (=) sign.
