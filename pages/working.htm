Working with AutoHotkey scripts

1. Running AutoHotkey scripts

Running AutoHotkey scripts can be done with a simple keypress (F5) or by clicking the Run button in the toolbar.

2. Compiling AutoHotkey scripts

Press F7 to compile your script. If you've installed Compile_AHK II, it'll be detected:

You can choose the compilation options, then click on Compile to compile the script.

3. Selecting the AutoHotkey version

If you have installed the Porting Pack, you can choose the AutoHotkey_L flavor you want to use for running and compiling scripts by clicking on the platform selection button in the toolbar ().
