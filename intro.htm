Introduction

SciTE4AutoHotkey is a SciTE-based AutoHotkey script editor. It offers:

  • Syntax highlighting
  • Calltips (also known as IntelliSense)
  • AutoComplete
  • AutoIndent
  • AutoHotkey help integration
  • Abbreviations
  • Editing macros
  • Debugging support
  • Tools for AutoHotkey scripting:
    • Active Window Info
    • SmartGUI Creator 4.0 (special version customized for this editor)
    • MsgBox Creator
    • GenDocs 3.0 (StdLib library documentation tool)
    • TillaGoto (script function/label/hotkey lister)
    • Scriptlet Utility (scriptlet management tool)
  • A toolbar that enables easy access to all of the above
  • Some AutoHotkey scripting facilities:
    • #Include file opener
    • Script folder opener
    • Create scriptlet from selected text

Neil Hodgson (the original author of SciTE) says the following about SciTE:

SciTE is a Scintilla based Text Editor. Originally built to demonstrate Scintilla, it has grown to be a generally useful editor with facilities for building and running programs.
