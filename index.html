What is it?

SciTE4AutoHotkey (screenshot) is a SciTE-based AutoHotkey script editor. It provides:

  • Syntax highlighting
  • Calltips (also known as IntelliSense)
  • AutoComplete
  • AutoIndent
  • AutoHotkey help integration
  • Abbreviations
  • Editing macros
  • AutoHotkey_L debugging support
  • Tools for AutoHotkey scripting
  • A toolbar that enables easy access to the tools

Based on PhiLho's AutoHotkey lexer for Scintilla, this provides a great script editor for both newbies and experienced AutoHotkey users.
